# Cyentic Studio - Official Website

Welcome to the official repository of **Cyentic Studio**. We design and build lightweight, functional offline-first utility applications and relaxing, high-quality casual games.

This web application serves as the landing page and developer portfolio for Cyentic Studio.

## 🚀 Live Demo
Visit our live website here: [https://Chitranjan-Sharma.github.io/ms-cyentic-studio-web/](https://Chitranjan-Sharma.github.io/ms-cyentic-studio-web/)

---

## 🛠️ Tech Stack & Architecture

- **Framework:** [React 18](https://react.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** Vanilla CSS (Custom design system with CSS custom properties)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Deployment:** GitHub Pages (via `gh-pages` branch)

---

## 🎨 Key Features

1. **Vibrant & Modern Design:** Rich gradients, card designs, and dynamic hover effects.
2. **Theme Switching:** Interactive dark/light mode toggle saved via `localStorage`.
3. **Responsive Layout:** Optimized for all screen sizes (mobile, tablet, desktop).
4. **Offline & App Values:** Showcases the studio's design philosophy—offline-first, zero trackers, clean layout.

## ⚙️ Development Guide

### Prerequisites
Make sure you have Node.js installed on your system.

### Installation
Clone the repository and install all dependencies:
```bash
npm install
```

### Run Locally (Development)
Start the Vite local development server:
```bash
npm run dev
```

### Build for Production
Generate optimized production builds in the `dist/` directory:
```bash
npm run build
```

### Deployment to GitHub Pages
To deploy the production build directly to GitHub Pages:
```bash
npm run deploy
```
